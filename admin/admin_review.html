GitHub Personal Access Token 등록

이 토큰은 본인 브라우저(localStorage)에만 저장됩니다. GitHub에는 절대 업로드되지 않아요.

발급 방법 (5분):

  1. github.com/settings/personal-access-tokens/new 이동
  2. Token name: artmap-review
  3. Expiration: 90 days
  4. Repository access: Only select repositories → artmap-sema-crawler
  5. Permissions → Repository permissions → Contents → Read and write
  6. 맨 아래 Generate token → 표시된 토큰 복사 (한 번만 보임)
